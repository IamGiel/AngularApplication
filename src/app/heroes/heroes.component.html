Data Binding

Below is an art work of data binding, where components handles functionality and controls the DOM. Two-way binding also is at work as both DOM and Component talks to each other and creates changes as user interact with the page. This is demonstrated by the input field.

The buttons are controlling the images using property binding, another one way binding. The button tag uses an event listener (on click) and calls a function declared inside the component.ts file.

(click) Handlers are implemented to listen to changes and are defined with a function that hides the other button via boolean. Moreover, we have used structural directives *ngIf (for switching a boolean from true to false and vice versa) and *ngFor (looping through array defined in the component).
