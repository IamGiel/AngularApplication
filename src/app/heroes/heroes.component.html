Data Binding

Below is an art work of data binding, where components handles functionality and controls the DOM. Dual binding also is at work as both DOM and Component talks to each other and creates changes as user interact with the page.

The buttons are controlling the images using property binding, another one way binding. The button tag uses an event listener (on click) and calls a function declared inside the component.ts file.
